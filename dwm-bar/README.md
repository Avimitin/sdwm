# dwm-bar

This is a simple program for showing information on the dwm bar.

## Features

Currently, it supports:

* Date and Time
* Battery
* Bluetooth Headset Battery
* Sound Volume
* Song Information

## Usage

```bash
cargo install --path .
~/.cargo/bin/dwm-bar &
```
